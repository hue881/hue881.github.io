🔍 Step 2 · What Is an Edge?

An edge is where numbers suddenly change! ⚡

🌅

In a photo

Where a dark sky meets a bright building — that sudden change is an edge. Your eye sees a line there!

🔢

In numbers

Going across a row:
0 → 0 → 5 → 5
The jump from 0 to 5 is an edge — a big number change!

🤖

Why AI needs it

If AI can find edges, it can see shapes. Shapes tell it: "this is a face", "this is a stop sign", "this is a cat"!

🧮 Step 3 · The Edge Formula

One simple idea: compare a pixel to its neighbors 🏘️

The big idea in plain English

For every pixel — ask:
"Are you very different from your neighbors?"

  • Big difference → you're on an edge! 🔴
  • Tiny difference → you're in a flat area ⬜
value = center×4 − (up+down+left+right)

Meet the 5 neighbors 👋

⬆️up
⬅️left
⭐
➡️right
⬇️down

⭐ = the pixel we're checking. The 4 arrows are its neighbors. If ⭐ is much brighter or darker than the arrows → EDGE!

🔑 Vocabulary · 4 Key Words

Learn these words — use them today! 📖

🖼️ Pixel

The tiniest dot in a digital picture. Each pixel stores one number. Zoom in on any photo and you'll see the squares!

📐 Edge

A place in the image where the numbers change a lot very quickly. Like the outline of a shape — where light meets dark.

🌀 Filter

A math operation that slides over every pixel and checks its neighbors. Different filters can blur, sharpen, or find edges.

🔁 Nested Loop

A loop inside another loop. The outer loop moves through rows, the inner loop through columns — visits every pixel one by one.

🐍 The Code · Line by Line

Let's read the code together — no scary parts! 😊

# Our tiny picture (5 rows, 5 columns)
image = [
  [0, 0, 0, 0, 0],
  [0, 5, 5, 5, 0],
  [0, 5, 9, 5, 0],
  [0, 5, 5, 5, 0],
  [0, 0, 0, 0, 0],
]

filtered = []  # our results go here

for y in range(1, 4):     # rows 1, 2, 3
  new_row = []
  for x in range(1, 4):   # columns 1, 2, 3
    center = image[y][x]
    up     = image[y-1][x]
    down   = image[y+1][x]
    left   = image[y][x-1]
    right  = image[y][x+1]

    # How different is center from neighbors?
    value = center*4 - (up + down + left + right)
    new_row.append(value)
  filtered.append(new_row)

print(filtered)

What each part does 🗺️

  • 📦 image — our 5×5 picture stored as a list of lists
  • 🔁 for y in range(1,4) — visits rows 1, 2, 3 (skips the edges!)
  • 🔁 for x in range(1,4) — visits columns 1, 2, 3
  • 👆 image[y-1][x] — reads the pixel above the current one
  • 📊 value — a big number means edge, small number means flat

💡 Why skip row 0? There's no row above it! y-1 would go off the grid.

🔢 Let's Calculate Together!

We do one pixel together — step by step ✍️

Pixel at row 1, column 1

center = image[1][1] = 5
up = image[0][1] = 0  ← row above
down = image[2][1] = 5  ← row below
left = image[1][0] = 0  ← column left
right = image[1][2] = 5  ← column right

value = 5×4 − (0+5+0+5)
= 20 − 10 = 10 🔴

10 is a big number — this pixel is on an edge! It sits right where the dark border meets the bright square.
