Just 4 Words to Remember (3 min)

Today's word kit 🔑

👍 Vote

A thumbs-up (1) or thumbs-down (0) from one person about one thing.

📊 Average

Add all the votes, then divide by how many people voted. Tells you "how much the group liked it."

🤖 Picker

The little program that does the picking for us. Other people call it a recommender — same thing.

🥇 Top

The item with the highest average. The picker shows the top first.

🧠 Quick check: If 5 friends thumbs-up a snack with [1, 1, 1, 1, 1], what's the average? What if it was [1, 0, 0, 0, 0]?

Pencil & Paper (5 min)

Find the average — by hand first ✋

Worked example: Racing 🏁

Five friends voted: [1, 1, 0, 1, 1]

Step 1. Add the thumbs-ups: 1+1+0+1+1 = 4

Step 2. Count voters: 5

Step 3. 4 ÷ 5 = 0.8

That's an 80% thumbs-up score 👍

?You try these

  • Puzzle: [1, 0, 0, 1, 0] → ?
  • Shooter: [0, 0, 1, 0, 0] → ?
  • Sports: [1, 1, 1, 1, 1] → ?

✋ Hold up the number with your fingers when you finish.

🥇 Which game has the biggest average? That's the one our picker will recommend first.

Code Step 1 of 3 (4 min)

Step 1 · Put the votes in a dictionary 📒

Type this in Python

ratings = { 'Racing': [1, 1, 0, 1, 1], 'Puzzle': [1, 0, 0, 1, 0], 'Shooter': [0, 0, 1, 0, 0], 'Sports': [1, 1, 1, 1, 1], } print(ratings)

?What's happening?

  • The name of each game is a key.
  • The list of votes is the value.
  • Together they're a dictionary: a labeled box of stuff.

📦 Run it. You should see all four games printed back at you. If not — raise your hand.

Code Step 2 of 3 (5 min)

Step 2 · Loop through and average each game 🔁

Add this below your code

averages = {} for game, votes in ratings.items(): score = sum(votes) / len(votes) averages[game] = score for game, score in averages.items(): print(game, '->', round(score, 2))

?Read it like a story

  • averages = {} — start an empty box for scores.
  • for game, votes in ratings.items() — visit each game one at a time.
  • sum(votes) / len(votes) — that's our average from slide 6.
  • round(..., 2) — keep just 2 decimal places so it's tidy.

▶️ Run it. Did Sports come out as 1.0?

Code Step 3 of 3 (3 min)

Step 3 · Pick the winner 🏆

Add one last line

best_game = max(averages, key=averages.get) print('Recommended first:', best_game)

Expected output

Racing -> 0.8 Puzzle -> 0.4 Shooter -> 0.2 Sports -> 1.0 Recommended first: Sports

?What does max do?

max finds the biggest. We tell it: biggest by which rule? By the score in averages.

🎉 You just wrote a real recommender! It's about 8 lines of code.

Talk Time (4 min)

When does "most popular" go wrong? 🤷

🍦

Tastes are different

If 19 kids love chocolate but YOU love mint, the popular pick isn't your pick.

🆕

Brand new things

A new flavor has zero votes. The picker can never put it on top — even if it's amazing.

🔁

Same thing again

If you only see the popular flavor, you might never try anything else.

💬 Class chat: Have you ever felt like an app only shows you the same kind of video over and over? Why might that happen?

Bonus Twist (5 min)

One picker per person 🧍

Try this in a fresh file

player_ratings = { 'Alex': {'Racing':[1,1,1], 'Puzzle':[1,1,0], 'Shooter':[0,0,0]}, 'Jordan': {'Racing':[0,1,0], 'Puzzle':[1,1,1], 'Shooter':[1,1,0]}, } for name, games in player_ratings.items(): print('\nFor', name) for game, votes in games.items(): score = sum(votes) / len(votes) print(' ', game, '->', round(score, 2))

?What changed?

  • Now Alex has his own ratings, and Jordan has her own.
  • Same average rule — but run once for each person.
  • Predict: who likes Racing more, Alex or Jordan?

▶️ Run it. Did each person get their own list? That's personalization — the secret behind YouTube.
