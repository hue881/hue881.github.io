Part 1 · v1 · Average-Rating Recommender

Code & verify 💻

Save the file as recommender_v1.py

Build it Type this exactly

ratings = { 'Racing': [1, 1, 0, 1, 1], 'Puzzle': [1, 0, 0, 1, 0], 'Shooter': [0, 0, 1, 0, 0], 'Sports': [1, 1, 1, 1, 1], } averages = {} for game, votes in ratings.items(): averages[game] = sum(votes) / len(votes) for game, score in averages.items(): print(game, '->', round(score, 2)) best_game = max(averages, key=averages.get) print('Recommended first:', best_game)

Part 2 · v2 · Personalized Recommender

One ranking per user 🧍🧍

Save as recommender_v2.py

Build it Nested dictionaries + sorted ranking

player_ratings = { 'Alex': {'Racing':[1,1,1], 'Puzzle':[1,1,0], 'Shooter':[0,0,0]}, 'Jordan': {'Racing':[0,1,0], 'Puzzle':[1,1,1], 'Shooter':[1,1,0]}, } for name, games in player_ratings.items(): print('\nRecommendations for', name) averages = {g: sum(v)/len(v) for g,v in games.items()} ranked = sorted(averages.items(), key=lambda item: item[1], reverse=True) for game, score in ranked: print(' ', game, '->', round(score, 2))

Part 4 · Stretch · Look-Alike Recommender

The Netflix Prize idea 🥇

Optional but powerful. ~10 min.

The challenge: Find Alex's most similar user, then recommend the one game Alex hasn't tried that the look-alike loved.

  1. Pick a similarity score. Easiest: count how many items two users agreed on (same 1 or same 0).
  2. Run the score for every other user.
  3. Pick the user with the highest similarity → that's the "look-alike".
  4. Recommend the look-alike's top item that Alex hasn't rated.
