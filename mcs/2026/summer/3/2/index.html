Mini-Lesson · Part 1

What is a Variable?

The Labeled-Box Idea

Variable Name
name
= "Mia"
Variable Name
score
= 42
Variable Name

= 3

In Python — what you type

# create a variable name = "Mia" score = 42 level = 3 # use it later print(name) # Mia print(score) # 42

Key rules: no spaces in the name · use = to assign · names are case-sensitive (Name ≠ name)

A variable is like a sticky-note on a box: the label (name) doesn't change, but you can swap out what's inside the box anytime!

Mini-Lesson · Part 3

String Formatting — Making Sentences with Variables

1

Concatenation
glue with +

print("Hi, " + name)

Watch out: can't + a string + a number directly!

2

Commas in print
easiest way

print("Hi,", name) print("Age:", age)

Works with any type — no error!

3

f-strings ✨
superpower move

print(f"Hi, {name}!") print(f"Age: {age}")

Put any variable between { } !

The letter f before the quote tells Python: "look for { } and fill them in with real values." f-strings are the most readable way to format output — use them as your default!

Vocabulary · 6 Key Words

Words every Python coder knows

Variable

A named container in memory. It stores a value that your program can use and change.

S
String

Text data — always inside quotes. "Hello", "42", and "🐍" are all strings.

#
Integer

A whole number with no quotes. score = 42 — integers can be added, subtracted, multiplied!

f"
f-string

A formatted string. Start with f"..." and put variable names inside { } — Python fills them in automatically!

⌨
input()

A built-in function that pauses the program and waits for the user to type something. It always returns a string!

▶
print()

A built-in function that displays text to the screen — your program's way of talking back to you!

🐛 Debug Corner · Common Bugs

Bugs you'll meet — and how to fix them

Bug 1: Forgot the quotes in a string

# ❌ WRONG name = Mia # NameError! # ✅ FIXED name = "Mia"

Without quotes, Python thinks Mia is a variable name, not text!

Bug 2: Can't add a string + a number

# ❌ WRONG print("Age: " + 12) # TypeError! # ✅ FIXED print("Age: ", 12) print(f"Age: {age}")

Use a comma or f-string to mix text and numbers!

Bug 3: input() gives you a string, not a number

age = input("Your age? ") print(age + 1) # TypeError! # ✅ FIXED age = int(input("Your age? "))

Wrap with int() to convert text → whole number

Bug 4: Variable used before it's created

# ❌ WRONG print(score) # NameError! score = 0 # ✅ FIXED — define first, use later score = 0 print(score)

Python runs top to bottom — define before you use!
