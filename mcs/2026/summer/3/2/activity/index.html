Key Ideas for Today
  • Variable — a named box in the computer's memory
  • String — text data, always inside quotes
  • Integer — a whole number, no quotes
  • input() — asks the user to type something
  • print() — shows something on the screen
  • f-string — fills variables into a sentence automatically
AI connection: A chatbot is just input → process → output — the same shape as today's Mad Libs! 🤖

Brain Break — Human Variable Game · 5 min

🧠 Stand up and become a variable!

Round 1 — String: Stand up and say your name out loud → name = "You"
Round 2 — Integer: Hold up fingers showing how many pets you have → pets = ?
Round 3 — Reassign: Swap seats with someone — you just changed the value of my_seat!
Bonus: Teacher calls print(name) — everyone shout your name at once!
You just became variables — a named container (you) holding a value (your name/number) that can be reassigned (swapping seats)!
