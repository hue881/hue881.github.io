Definitions

A. A value that is only TRUE or FALSE
B. A block that only runs code if a condition is true
C. A path a program can take depending on a condition
D. A rule that makes a program choose between two or more actions
E. A diagram showing all possible decisions and their outcomes — like how AI classifiers work
