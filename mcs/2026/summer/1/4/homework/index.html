🔑 Key Ideas to Remember
  • Loop — do something many times without writing it many times
  • repeat N stops after exactly N runs; great for shapes
  • forever runs until Stop; great for animation and games
  • 4 sides × 90° = 360° = a complete square
  • Training loop — AI repeats through examples until it learns

🔵 Tier B — Core · Grades 3–5
🎯 Your Mission

Build a repeating animation or pattern that uses both repeat and forever in the same project. Count the code savings!

Core build — pick one, or combine!

  • Repeating spiral: increase move steps by 5 each time inside repeat 20
  • Bouncing dancer: forever → move + bounce + switch costume + wait
  • Clock hand: repeat 360 → turn 1 degree + wait 0.01 sec → full rotation!
# Spiral pattern idea: repeat 20 move steps ← increase by 5 each time turn 90 degrees
🏁 Checklist
  • Project uses repeat N somewhere
  • Project uses forever somewhere (or both scripts use repeat)
  • I counted the code lines a loop saved me (see Part 3)
  • I described what my project does in one sentence
  • Saved my project
💡 Spiral tip: In Scratch, you can't auto-increase a variable inside a loop without using a variable block — try adding a "step size" variable that grows each loop!
⭐ Bonus Tier B: Make your sprite say how many times the loop ran using a "say [ ] for 2 seconds" block after the loop ends.
🟢 Tier C — Challenge Card · Grades 6–9
🎯 Your Mission

Build a nested loop generative art piece or a loop + variable animation. Your project should look different every time (or be more complex than today's class build).

Challenge options (pick one or combine):

  • Random color mandala: outer repeat 12 → inner repeat 4 (square) → turn 30° + random pen color each outer loop
  • Growing spiral: repeat 80 → move step steps → turn 91° → change step by 2
  • Star burst: repeat N → move forward → turn (360÷N)° — try different N values
# Growing spiral (use a variable): set [step] to 2 repeat 80 move (step) steps turn 91 degrees change [step] by 2

🤖 Part 4 of 4 · AI Reflection ⏱ 5 min
Connect Your Loop to Real AI
🧠 Read · Think · Write

Here's how AI really trains:

# Inside every AI (simplified): repeat 10000 times: look at one training example make a prediction check: was it right? if wrong → adjust a tiny bit # after 10,000 loops → smarter AI!
🔁 This is called an epoch in real AI — each pass through all the training data is one epoch. A model might train for 50+ epochs!

Real AI models like image classifiers can loop through millions of examples, thousands of times. Your computer's job is just like your Scratch loop — but much, much bigger!

✍️ Write 2–3 sentences for each

1. How is your Scratch loop similar to how AI trains?

2. If an AI ran its training loop only 3 times instead of 10,000, what would happen and why?

3. (Tier B + C) In your own words: what is an "epoch" and how does it relate to repeat N?
