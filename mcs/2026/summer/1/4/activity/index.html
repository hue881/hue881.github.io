🔑 Key Ideas for Today
  • Loop — do something many times without writing it many times
  • repeat N — runs the inside exactly N times, then stops
  • forever — keeps running until you hit the Stop button
  • Fixed count → use repeat N   ·   Never-ending → use forever
  • Training loop — how AI gets smarter by repeating practice!
💡 Predict before you build — if you can say what the code will do before running it, you truly understand the loop!

How Does AI Practice? It Loops!
🧠 The Connection: Your Code = AI's Training Engine

When an AI learns to recognize cat photos, it doesn't look at each photo once. It loops through thousands of cat photos — over and over — getting a little better each time.

# How AI "trains" (pseudocode): repeat 1000 times: look at one cat photo make a guess check if wrong adjust → gets smarter!
🔁 More loops = more practice = smarter AI. Just like practicing piano — once is not enough!
