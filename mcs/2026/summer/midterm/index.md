#### ✍️ Question Types in Part 1 (CAT — Computer Adaptive Test)

The digital exam uses a **15-question adaptive format**:

- **Multiple choice (10 questions)** — pick the best answer from 4 options
- **Matching (5 questions)** — match terms to their definitions (e.g., "loop → code that repeats")
- **Difficulty levels 1–5** — the test adjusts up or down based on each answer
- **Ability meter** — a live θ (theta) score from –3 to +3 tracks student level
- **Scaled score** — converted to 200–800 range (SAT-style); speed on correct answers gives a small bonus

- 💡 **Teacher note:** The adaptive engine starts at difficulty 3 (middle), raises by ~0.28×difficulty step on correct, lowers by ~0.28×difficulty step on wrong. After 15 items, it assigns a **Tier** (A, B, or C) that routes Part 2.

## 🤖 AI Integration Thread (Session 11)

Every session has an embedded AI moment — the midterm is no exception.

> **AI Section of the exam:** Students are asked to:
> 1. **Explain what AI is** in their own words
> 2. **Give one example of AI being wrong** (responsible-AI check)
> 3. (Tier B/C) Connect **training data → model → prediction** pipeline
> 4. (Tier C) Explain **bias** — when limited/unfair data causes unfair results

This directly maps to the **AI Integration Map** for Sessions 8–14:
> *"Text & data: chatbots as input → output, datasets as lists, search strategies, rules vs. learning"*

Key AI vocabulary tested:

| Term | Plain-Language Meaning |
|------|----------------------|
| **Training data** | Examples an AI learns from (like pictures of cats & dogs) |
| **Model** | The pattern the AI built after studying the examples |
| **Prediction** | The AI's best guess based on the pattern |
| **Confidence** | A score (like 87%) — how sure the AI is about its guess |
| **Bias** | When unfair or incomplete data makes an AI unfair or inaccurate |
| **Rule-based AI** | Follows rules a human wrote (like our `if/else` chatbot) |
| **Learned AI (LLM)** | Learned patterns from millions of examples by itself |



### Part 1 CAT Engine

```
Start at θ = 0 (difficulty 3)
  ↓
Answer question → correct?
  ↓ YES: θ += 0.28 × (difficulty - 3) + 0.045
  ↓ NO:  θ -= 0.28 × (difficulty - 3) + 0.025
After 15 items → Compute Tier (A/B/C) & Scaled Score (200–800)
```

**Scaled Score formula:**
- Base: `500 + θ × 100` (range 200–800)
- Speed bonus: faster correct answers add up to ±40 points
- Rounded to nearest 10 (SAT-style)

**Tier cutoffs:**
- Tier C: correct ≥ 12 **and** θ ≥ 0.45
- Tier B: correct ≥ 8 **and** θ ≥ –0.45
- Tier A: everyone else (fewer correct or lower θ)

### Part 2 Adaptive Selection

- 7 questions chosen by weighted difficulty, targeting the student's Part 1 ability level
- 2 "mixed" slots intentionally set 1–1.5 levels below target (for balance and confidence)
- Remaining 5 from weighted target difficulty pool

### Timers
- **Part 1:** 20 minutes (1,200 seconds) — timer turns yellow < 5 min, red < 2 min
- **Part 2:** 80 minutes (4,800 seconds)

### Scoring Formula

Final Part 2 Score = (Project Checklist % × 0.70) + (MC Quiz % × 0.30)

## 💡 Key Code Snippets Students Should Know

### Python — Tier B/C Level

```python
# Variables and input
name = input("What is your name? ")
print("Hello, " + name + "!")

# For loop with range
for i in range(3):
    print("Hi!")
# Output: Hi! Hi! Hi!

# Loop with counting
score = 2
for i in range(3):
    score = score + 1
print(score)
# Output: 5

# Counting with a conditional inside a loop
count = 0
for animal in ["cat", "dog", "cat"]:
    if animal == "cat":
        count = count + 1
print(count)
# Output: 2

# turtle drawing (introduced Session 10)
import turtle
for i in range(4):
    turtle.forward(100)
    turtle.right(90)
```

### Scratch — Tier A/B Level

```
// Sequence + Event
when green flag clicked
  move 10 steps
  say "Hello!" for 2 seconds

// Loop
when green flag clicked
  repeat 3
    say "Hi!" for 1 second

// Conditional with variable
when green flag clicked
  set score to 0
  forever
    if touching "coin" then
      change score by 1
```
