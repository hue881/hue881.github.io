🛠️ Part 3 of 6 · Guided Build (20 min) — Everyone Together
Build One Clean Round of Rock-Paper-Scissors
📋 Type this along with your teacher, step by step
import random moves = ['rock', 'paper', 'scissors'] player = input("Pick rock, paper, or scissors: ") computer = random.choice(moves) print("Computer picked:", computer) if player == computer: print("Tie!") elif (player == 'rock' and computer == 'scissors') or \ (player == 'paper' and computer == 'rock') or \ (player == 'scissors' and computer == 'paper'): print("You win!") else: print("Computer wins!")
  • My moves list has exactly 3 items
  • The computer's move is different most times I run it
  • Tie, win, and lose all print the correct message when I test them
