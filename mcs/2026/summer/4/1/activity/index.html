What this activity sheet is for
Daily Rhythm Match

This sheet follows the class rhythm from the syllabus: warm-up, mini-lesson, guided build, movement break, open build, and share-out. Everyone works on the same concept, but each tier has a different on-ramp.

Today’s goal: use Python turtle and a for loop to make code draw shapes and spirals.
Lesson ideas from class
  • Bridge from Scratch: repeat blocks become for i in range(...).
  • Visual output: turtle makes Python feel concrete and beginner-friendly.
  • AI tie-in: generative art uses rules plus a little randomness.
Part 1 · 10 min
Warm-Up · Predict the pattern
No-screen thinking first

Look at each tiny code idea and predict what the turtle would draw. You do not need to run it yet.

for i in range(4): t.forward(60) t.right(90)
for i in range(3): t.forward(80) t.left(120)
Think aloud: How many times does the loop repeat? What turn angle helps the shape close?
Part 2 · 15–20 min
Mini-lesson check · Read the code
Five commands to know
CommandWhat it does
import turtleLoads the drawing library.
t = turtle.Turtle()Creates the turtle object.
t.forward(n)Moves forward n pixels.
t.right(a)Turns right by a degrees.
t.pencolor(...)Changes pen color.
Label the parts
import turtle t = turtle.Turtle() t.pencolor("blue") for i in range(4): t.forward(70) t.right(90) turtle.done()

Part 3 · 20 min
Guided build · Make the class spiral
Build together first
import turtle import random t = turtle.Turtle() t.speed(0) colors = ["red", "orange", "blue", "green"] for i in range(100): t.pencolor(random.choice(colors)) t.forward(i) t.right(91) turtle.done()

Trace the loop: fill in the first four turns.

Explain the spiral: Why does the picture grow instead of closing like a square?

AI moment: Each run can look a little different because of random.choice().

Part 4 · 5 min
Movement break · Be the turtle
Quick brain break
  • Step forward 3 steps.
  • Turn right 90 degrees.
  • Repeat 4 times. What path did your body make?
  • Now try 1 step, 2 steps, 3 steps, 4 steps. What changes?
Part 5 · 30–40 min
Open build · Choose your tier
Tier A · Foundations

Goal: draw one clear shape with a loop.

  • Use forward, right or left, and one for loop.
  • Make a square or triangle.
  • Change the pen color once.
  • Optional: use penup() and pendown() to move and draw a second shape.
  • I used a loop.
  • My shape closes correctly.
  • I can point to the turn angle I used.
    • Tier B · Core

    Goal: build a colorful spiral like the lesson model.

    • Use for i in range(...).
    • Use forward(i) or another growing value.
    • Add a list of colors and random.choice().
    • Run it twice and compare the results.
    Tier C · Accelerated

    Goal: design geometric art with a stretch challenge.

    • Use a loop inside a loop, or create multiple turtles.
    • Experiment with angle, pen width, and background color.
    • Add one comment that explains your algorithm.
    • Make an intentional art choice, not just a random picture.
