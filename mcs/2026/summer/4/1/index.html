Mini-Lesson · Part 1

What is turtle?

The Big Picture

turtle comes from a real robot invented in 1967 that dragged a pen across the floor. Python's turtle does the same thing — on your screen! It's already built into Python — no install needed.

The Scratch ↔ Python Bridge

Scratch: move 10 steps → Python: t.forward(10)
Scratch: turn right 90° → Python: t.right(90)
Scratch: repeat 4 → Python: for i in range(4):
Scratch: set pen color → Python: t.pencolor("blue")

Notice: You already know how turtle works — you used it in Scratch! Today you're just writing the same instructions in Python text instead of dragging blocks.

Mini-Lesson · Part 2

Anatomy of Your First Turtle Program

import turtle          # load the library

t = turtle.Turtle()    # create your turtle

t.pencolor("blue")    # choose a color
t.forward(100)         # move forward
t.right(90)            # turn right
t.forward(100)

turtle.done()          # keep window open

What does it draw? Two sides of a square. What do you need to add to finish the square?

5 Commands to Know Today

1t.forward(n) — draw a line of n pixels
2t.right(a) — turn right by a degrees
3t.pencolor("red") — change color
4t.penup() / t.pendown() — lift or place pen
5t.speed(0) — go as fast as possible!

Try online (no install): trinket.io/turtle — it runs turtle right in the browser!

Mini-Lesson · Part 3

The for Loop — Decoded

Reading the Code

for i in range(4):
    t.forward(100)
    t.right(90)
for i in range(4): — "repeat 4 times, counting i from 0 to 3"
indented lines — everything indented runs each time the loop repeats
result: a perfect square! (4 sides × 90° = 360°)

Using i as a Variable Inside the Loop

for i in range(100):
    t.forward(i)   # grows each time!
    t.right(91)    # slightly off 90°

Here i starts at 0 and goes up to 99. Each step the turtle moves a little farther — that's why the spiral grows!

What happens if you change 91 to 90? What about 89? What about 120? Predict first, then try!

Scratch connection: In Scratch, repeat 4 is the same as Python's for i in range(4). The difference? In Python, you can use the counter variable i inside your loop!

Phase 2 · Sessions 8–14

Generative Art: Rules + Randomness = Surprise!

Add Randomness with random

import turtle
import random

t = turtle.Turtle()
colors = ["red", "blue", "green", "gold"]

for i in range(100):
    t.pencolor(random.choice(colors))
    t.forward(i)
    t.right(91)

Run it twice. Does it draw the same thing? Why not?

This IS how Generative AI works!

When an AI tool draws an image, it follows rules (patterns it learned from millions of pictures) and adds randomness (sampling). Each run is unique — just like your turtle spiral!

The key difference

Your turtle follows rules you wrote. A generative AI follows rules it learned from data. Both mix rules + randomness to create something new every time.

Question: Could you write a rule that always made beautiful art? Or does it need some randomness to feel creative?

Debug Corner · Common Bugs

Bugs are normal — let's find them together!

Bug 1 — Forgot to indent

for i in range(4):
t.forward(100)  # ← NO indent = ERROR

Python uses indentation to know what's inside the loop. Always indent with 4 spaces (or Tab) after the colon!

Bug 2 — Forgot the colon

for i in range(4)  # ← missing :
    t.forward(100)

Every for line ends with a colon :. Think of it as Python saying "here comes the block!"

Bug 3 — Window closes instantly

Did you forget turtle.done() at the end? Without it, the window closes as soon as drawing finishes. Add it as the very last line!

Bug 4 — random not imported

# Missing at the top:
import random   # ← add this!

If you use random.choice(), Python needs you to import the random library first — always at the very top.

Pro move: Read the error message carefully — Python tells you exactly which line has the problem and what went wrong!

Vocabulary · 6 Key Terms

Words every Python artist knows

📦
Library / Module

A collection of ready-made tools you can import. turtle and random are standard library modules — free and always available in Python.

🔁
for Loop

Repeats a block of code a set number of times. for i in range(n) runs n times, with i counting up from 0.

↔️
Iteration

One trip through a loop. A loop with range(100) has 100 iterations.

🎲
random.choice()

Picks one item at random from a list. Pass it a list: random.choice(["red","blue"]).

🐢
Turtle object

The pen you control. t = turtle.Turtle() creates your turtle, then you call its methods: t.forward(), t.right(), etc.

🎨
Generative Art

Art created by rules + randomness in code. Every run is unique. Your turtle spiral is generative art — and so are AI-generated images!
