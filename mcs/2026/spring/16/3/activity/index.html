🔑 Key Formula

Every answer today uses the Laplacian-style edge filter:

value = center × 4 − (up + down + left + right)
  • Big positive value → edge detected 🔴
  • Value near zero → flat region, no edge
  • We skip row / col 0 and the last row / col

# Part A: Define a 5×5 grayscale image image = [ [0, 0, 0, 0, 0], [0, 8, 8, 8, 0], [0, 8, 8, 8, 0], [0, 8, 8, 8, 0], [0, 0, 0, 0, 0], ] # Part B: Apply the Laplacian edge filter filtered = [] for y in range(1, 4): # skip row 0 (no above) and row 4 (no below) new_row = [] for x in range(1, 4): # skip col 0 and col 4 center = image[y][x] up = image[y-1][x] # one row above down = image[y+1][x] # one row below left = image[y][x-1] # one column left right = image[y][x+1] # one column right value = center*4 - (up + down + left + right) new_row.append(value) filtered.append(new_row) # Part C: Print results print("Filtered grid:", filtered) # Part C+: Label edges for row in filtered: for v in row: if v > 10: print("EDGE!", v) else: print("flat", v)
