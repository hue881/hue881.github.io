One-Neuron Trainer

Code a single neuron that learns to classify numbers as big or small. Train it with the weight-update rule and watch it improve.

Day 14Python / JSIntermediate
Steps:
  1. Set up w, b, and a learning rate
  2. Build predict(): raw = w*x + b, return 1 if raw > 0.5
  3. Write the training loop with error = target - pred
  4. Test on at least 10 examples
  5. Challenge: visualize how w changes each step

Recommendation System

Build a system that recommends games, movies, or songs based on simulated friend ratings. Display rankings from highest to lowest.

Day 15Python / JSBeginner
Steps:
  1. Create a list of at least 6 items with ratings from 5 friends
  2. Compute average rating for each item
  3. Sort and display recommendations in order
  4. Add a way for the user to add their own rating
  5. Challenge: filter by category or friend preference
🔍

Edge Detector

Apply the Day 16 filter formula to a pixel grid and highlight edges. Visualize the original grid and the edge map side by side.

Day 16Python / JSIntermediate
Steps:
  1. Create a 5×5 or larger pixel grid with varied values (0–9)
  2. Apply: value = center*4 - (up + down + left + right)
  3. Display original and edge maps as colored tables
  4. Mark high-value cells as "edges" with a highlight color
  5. Challenge: let the user draw their own pixel grid
🗺️

Maze Solver Robot

Build an animated DFS maze solver. Draw the grid, show the robot stepping through it, and highlight the visited cells and final path.

Day 13JavaScriptIntermediate
Steps:
  1. Create a 5×5 grid with walls (0 = wall, 1 = open)
  2. Implement DFS with a stack and visited set
  3. Animate each step with a delay (setTimeout)
  4. Color visited cells blue, the path green, walls gray
  5. Challenge: let the user design their own maze
