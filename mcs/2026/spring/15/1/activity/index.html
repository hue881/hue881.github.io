Part 1 · Code · Step-by-step

Type along — 3 small steps 💻

Open your Python editor.

Step 1 Make the votes dictionary

Type this exactly. Then click Run.

ratings = { 'Racing': [1, 1, 0, 1, 1], 'Puzzle': [1, 0, 0, 1, 0], 'Shooter': [0, 0, 1, 0, 0], 'Sports': [1, 1, 1, 1, 1], } print(ratings)

What do you see printed?

Step 2 Loop and average each game

Add this below what you already typed.

averages = {} for game, votes in ratings.items(): score = sum(votes) / len(votes) averages[game] = score for game, score in averages.items(): print(game, '->', round(score, 2))

Write the four lines that printed:

Step 3 Pick the winner

Add these last two lines.

best_game = max(averages, key=averages.get) print('Recommended first:', best_game)

Expected output:

Puzzle -> 0.4
Shooter -> 0.2
Sports -> 1.0
