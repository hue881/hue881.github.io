📋 What to do

Convert TinyBot v1's if/elif chain to the data-driven dict structure. Every reply must still work. Adding a new intent should only require a new dict entry — no new code logic.

  • Fill in the REPLIES dict with matching responses
  • Complete get_intent() using set intersection
  • 4 intents in RULES (greeting, farewell, negative, positive)
  • Matching REPLIES for each intent + "unknown"
  • get_intent() uses set() — not in substring check
  • Adding "weather" intent takes under 60 seconds ✓
